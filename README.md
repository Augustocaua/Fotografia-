# ACAUS — Página de Vendas de Fotografia

Projeto de landing page para divulgação de serviços de fotografia de casamento.

- Assinatura do projeto: ACAUS
- Tecnologias: Vite, React, TypeScript, Tailwind CSS, shadcn-ui

## Como rodar localmente

Pré-requisitos: Node.js e npm instalados.

```sh
npm install
npm run dev
```

Acesse o preview em `http://localhost:8080/`.

## Build e Preview de Produção

```sh
npm run build
npm run preview
```

## Estrutura do Projeto

- `src/components/` — Componentes React (Hero, Services, Gallery, About, Testimonials, Contact, Footer)
- `src/assets/` — Imagens e recursos estáticos
- `src/index.css` — Estilos globais
- `tailwind.config.ts` — Configuração Tailwind
- `vite.config.ts` — Configuração Vite

## Padrões de Código

- TypeScript com React 18
- Estilização com Tailwind e shadcn-ui
- Alias de imports: `@` aponta para `src`

## Assinatura e Créditos

Projeto desenvolvido pela ACAUS para portfólio.
